Befehle im Menü 'Datei'

Das Menü Datei enthält folgende Befehle:

Neu Erstellt ein neues Dokument.
Öffnen Öffnet ein vorhandenes Dokument.
Schließen Schließt ein Dokument.
Speichern Speichert ein geöffnetes Dokument unter dem gleichen Dateinamen.
Speichern unter Speichert ein geöffnetes Dokument unter einem angegebenen Dateinamen.
Beenden Beendet <<YourApp>>.
